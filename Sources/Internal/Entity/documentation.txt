DAVA FRAMEWORK ENTITY SYSTEM DOCUMENTATION
=================================================

У нас есть 3 типа объектов. 
1) Пул с данными. 


2) Компонент


3) Система


Несколько потоков данных в компоненте, дают нам возможность автоматически сортировать
